# VoiceBox

VoiceBox is a writing tool that lets you create named "voice" style
profiles and rewrite AI-generated drafts through them, producing output
that sounds like you (or any style you define) instead of generic AI
text.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
